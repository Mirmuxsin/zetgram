# Zetgram 
PHP Framework for create Telegram bot simple.

# EN:
Download and install
---------
  #### With Github
  Run this in catalog:
  ```
  https://github.com/zetgram/zetgram.git
  ```

  #### Run this with composer:
  ```
  composer require zetgram/zetgram
  ```
